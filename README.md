# Spam Message Detector

This project is a Machine Learning-based application that classifies messages as **Spam** or **Not Spam (Ham)** using Natural Language Processing (NLP).

## Features
- Detects spam messages with good accuracy
- Simple and fast prediction
- Uses trained machine learning model
- Easy to understand and use

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy
- NLP (TF-IDF Vectorization)

## Project Files
- `app.py` – Main application file
- `model.py` – Model training script
- `model.pkl` – Trained ML model
- `vectorizer.pkl` – Text vectorizer
- `spam.csv` – Dataset

## How It Works
1. Input message is cleaned and processed  
2. Text is converted into numerical form using TF-IDF  
3. Trained model predicts whether the message is spam or not
